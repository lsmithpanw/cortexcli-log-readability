# cortexcli-log-readability
Human-friendly cortexcli log output

This tool outputs logs from the last Cortex Cloud [cortexcli](https://docs-cortex.paloaltonetworks.com/r/Cortex-XDR/Cortex-XDR-4.x-Documentation/Cortex-CLI-common-command-line-reference-guide) image scan run in a human-readable way.

## Usage
`./cortexcli image scan nginx:latest`  
`./ccl.sh`
